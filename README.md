# ReactNative TypeScript
> Initialize typescript into your react native project

## How to initialize?
```sh
$ npm i -g rn-ts
$ cd /PATH/TO/YOUR/RN_PROJECT
$ rnts i
```

## How to run?
```sh
$ npm run build:ts
```

##⚠️ CAUTION##
You **MUST** install `@types@/react` and `@types/react-native` to run `build:ts` command.
